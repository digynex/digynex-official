# Elite Tuition Manager (TMS) - Case Study
**By DigyNex (Amila Perera - Enterprise Automation Architect)**

---

## 🚀 1. Executive Summary
**The Client:** A highly respected educational institution struggling with manual data entry, scattered financial records, and inefficient student communication.
**The Challenge:** The academy depended on a mix of Excel sheets and WhatsApp groups. Tracking fee payments, managing attendance, and notifying parents took up to 20 hours of administrative work per week.
**The Solution:** An End-to-End Enterprise Management SaaS built with **Vue.js**, **Supabase**, and automated via **n8n** & **Generative AI**.

---

## 🛠️ 2. The DigyNex Tech Architecture 
We didn't just build a basic script; we engineered a scalable ecosystem.

*   **Frontend (UI/UX):** Custom-built using **Vue.js** & **Tailwind CSS**. Features a premium dark-mode interface, interactive chart dashboards, and lightning-fast glassmorphism components.
*   **Backend & Database:** Powered by **Supabase (PostgreSQL)** ensuring Row Level Security (RLS) and real-time data synchronization across all administrative screens.
*   **Workflow Orchestration:** Enterprise-grade automation using **n8n**. 
*   **AI Integration:** Powered by the **Antigravity Engine**. Features automated invoice scanning and conversational routing using AI.

## 💥 3. Key Implementations & Features
*(Note for Design: Add 1-2 screenshots of the dashboard here)*

### A. Intelligent Financial Dashboard
We replaced fragmented Excel sheets with a centralized dashboard. The system automatically tracks pending payments, generates real-time revenue graphs (using Chart.js), and highlights financial anomalies instantly.

### B. Automated Attendance & WhatsApp Alerts
No more manual texting. Upon student check-in, the system triggers an **n8n workflow** that instantly sends a branded WhatsApp notification to the parents, confirming attendance securely.

### C. AI-Powered Expense Management
Staff members simply upload a picture of a receipt. Our **n8n + Gemini AI pipeline** automatically reads the image (OCR), categorizes the expense, extracts the exact amount, and logs it directly into the Supabase database.

## 📊 4. The Impact (Results)
*   **85% Reduction in Admin Time:** What took 20 hours a week is now completely automated.
*   **Zero Data Loss:** Transitioning from local Excel files to secure cloud storage (Supabase) guaranteed data integrity.
*   **100% Communication Accuracy:** Automated WhatsApp routing ensured parents always received timely updates without human error.
