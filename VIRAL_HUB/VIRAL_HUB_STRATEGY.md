# DigyNex Viral Intelligence Hub (VIH) | Complete Strategic Roadmap

## 📋 Initial Agreement & Project Vision

The objective is to build a standalone, professional-grade viral content architect. This system will serve as a high-end selling point for clients, automating the discovery of viral trends and the generation of creative assets.

### Initial Objectives:

1. Identify 10 high-growth viral niches using predictive AI.
2. Provide professional-grade image prompts optimized for Midjourney and Kling.
3. Generate cinematic auditory scripts and sound effect (SFX) metadata.
4. Establish a modular, clean code architecture (SOLID/DRY) for easy scalability.

---

## 🛠️ Requested Adjustments (Refinements based on User Feedback)

The following specific changes must be implemented into the system logic and UI:

### 1. Engine Branding Update (Gemini Latest)

- **Problem:** Referring to "Gemini 1.5" creates a technical ceiling and limits future scalability/client perception.
- **Requirement:** Standardize all internal and external terminology to **"Gemini Latest Engine"**. Ensure this reflects a commitment to using the highest-tier AI models available.

### 2. Multi-Language Support

- **Problem:** Global reach requires accessibility in multiple regions.
- **Requirement:** Implement a front-end toggle for three core languages: **English, Sinhala, and Swedish (SV)**. All dashboard labels and generated outputs must reflect the selected language.

### 3. Professional-Grade Prompt Engineering

- **Problem:** Short, simple sentences are insufficient for high-quality viral outputs.
- **Requirement:** The AI engine must generate highly detailed, technical, and professional prompts. This includes:
  - Dedicated Scene Locking metadata (Camera lens, focal length, tripod settings).
  - Detailed lighting and texture descriptions.
  - Negative prompts to ensure high-fidelity results.
  - Multi-stage consistency (Stage 1-4) for seamless timelapse transitions.

### 4. Advanced Auditory Design

- **Problem:** Sound suggestions need to match the professional quality of the visuals.
- **Requirement:** Generate complex, atmospheric soundscape designs and clear, professional voiceover scripts for every niche.
